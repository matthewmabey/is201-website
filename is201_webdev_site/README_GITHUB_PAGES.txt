How to publish:
1. Create a new GitHub repository.
2. Upload every file in this folder.
3. In the repo settings, open Pages.
4. Set the source to the main branch and root folder.
5. Wait a minute for GitHub Pages to publish.
6. Open the live URL and take screenshots of each page with the address bar visible.
7. Paste those screenshots into the Word document template.
